Code For Nature

Java SQL Java Swing

This project was developed as an assignment for the Fundamentals of Programming course. It focuses on raising public awareness about environmental issues such as climate change, pollution, and deforestation. The project features an interactive knowledge platform where users can take daily quizzes to enhance their understanding of nature-related challenges. A point-based reward system was implemented to encourage consistent user engagement, allowing points earned through quizzes to be redeemed for nature-care merchandise or tree-planting initiatives. The project also includes a donation feature for users to contribute directly to authorized environmental organizations. We developed a user-friendly GUI, connected the platform to an SQL database, and implemented password hashing for improved security. Through this combination of education, gamification, and accessibility, the project aims to promote conservation efforts and inspire users to take action.
